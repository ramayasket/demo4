# demo4
Тестовое задание для UNADESK

Условия: https://docs.google.com/document/d/1GKmRARVykfB7zqTW74_ISGR9RqFYJg3wR74pZLCX3Do/edit?tab=t.0

Автор: Самойлов Андрей

Комментарии:
============

1) Программа сделана в минималистском характере, как демонстратор.
   Не реализованы: обработка ошибок (кроме парсинга PDF), тайм-аут ожидания ответа от сервера и т.п.

   Текущая структура очередей позволяет иметь 1 экземпляр клиента (Api) и N экземпляров сервера (Worker).

2) Программа использует авторскую библиотеку **Kwisatz.Haderach** для работы с очередями, асинхронного ожидания ответа от сервера и др. функций.

   Пакеты:
   - **Kw.Common** (общие многоцелевые классы)
   - **Kw.Micro** (общие классы микросервисов)

3) Для работы с PDF-документами используется пакет **PdfPig**.
